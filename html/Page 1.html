SQL Server


SQL Server es un sistema de gestión de bases de datos relacional desarrollado por Microsoft, que permite almacenar, recuperar, administrar y proteger datos de manera eficiente y segura en una variedad de aplicaciones y entornos empresariales. Está basado en el lenguaje Transact-SQL e incorpora un conjunto de extensiones de programación propias de lenguaje estándar y su aplicación está disponible para usarse tanto a nivel on premise o bajo una modalidad cloud.con la información de otros grupos.

SQL Server dispone de los siguientes servicios:
  • Agente SQL Server, es un servicio de Microsoft Windows que ejecuta tareas administrativas programadas, denominadas trabajos, en SQL Server.
  • Servicio SQL Server Browser, es un programa que se ejecuta como un servicio de Windows. SQL Server Browser escucha las solicitudes entrantes de Microsoft recursos de SQL Server y proporciona información acerca de las instancias de SQL Server instaladas en el equipo.
  • SQL Server Integration Services (SSIS Server), es una plataforma para la creación de soluciones empresariales de transformaciones de datos e integración de datos. Use Integration Services para resolver complejos problemas empresariales mediante la copia o descarga de archivos, la carga de almacenamientos de datos, la limpieza y minería de datos y la administración de datos y objetos de SQL Server.
  • SQL Server Analysis Services (SSAS Server), es un motor de datos analíticos (VertiPaq) que se usa en el soporte para la toma de decisiones y el análisis empresarial. Proporciona modelos de datos semánticos de nivel empresarial para informes empresariales y aplicaciones cliente como Power BI, Excel, informes de Reporting Services y otras herramientas de visualización de datos.
  • SQL Server Reporting Services (SSRS Server), ofrece un conjunto de servicios y herramientas locales que permiten crear, implementar y administrar informes paginados. Asimismo, se puede usar para desarrollar aplicaciones de informes.

La versión más reciente es Microsoft SQL Server 2022 (1.6x), y esta disponible en las siguientes 5 versiones:

  1. Enterprise
  2. Estandár
  3. Web
  4. Desarrollador
  5. Express
